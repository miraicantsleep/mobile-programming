Marketplace Siswa

Aplikasi Android sederhana berbasis Jetpack Compose yang memfasilitasi siswa untuk melihat dan menjual produk di lingkungan sekolah. Proyek ini merupakan implementasi studi kasus kelas Pengembangan Mobile berdasarkan referensi:

  • Blog: Material Studi Kasus Marketplace Siswa
  • Spesifikasi PRD: Modul Material Design & Modern Android UI/UX
  • Kode referensi: Marketsiswa.pdf

Penjelasan Kode Per File

data/Product.kt

Mendefinisikan data model produk yang digunakan di seluruh aplikasi.

data class Product(
    val id: Long = System.currentTimeMillis(), // ID unik berbasis timestamp
    val name: String,
    val price: String,      // Disimpan sebagai String agar fleksibel ditampilkan
    val description: String
)

val seedProducts = listOf(
    Product(id = 1L, name = "Brownies Lumer", price = "15000", description = "..."),
    Product(id = 2L, name = "Kaos Custom",    price = "85000", description = "...")
)
  • data class – Kotlin idiom untuk class yang tugasnya hanya menyimpan data. Otomatis mendapat equals(), hashCode(), dan copy().
  • id: Long = System.currentTimeMillis() – Nilai default menggunakan timestamp saat objek dibuat, sehingga setiap produk baru mendapat ID unik tanpa database.
  • seedProducts – Data awal yang langsung muncul saat aplikasi pertama kali dibuka (Brownies Lumer & Kaos Custom sesuai kode referensi).

viewmodel/MarketplaceViewModel.kt

Lapisan ViewModel bertanggung jawab atas state daftar produk dan operasi penambahan.

class MarketplaceViewModel : ViewModel() {
    val products = mutableStateListOf<Product>().apply { addAll(seedProducts) }

    fun addProduct(name: String, price: String, description: String) {
        products.add(0, Product(name = name, price = price, description = description))
    }
}
  • ViewModel – Bertahan selama rotasi layar (configuration change), sehingga data tidak hilang saat HP diputar.
  • mutableStateListOf – List yang observable oleh Compose. Setiap perubahan memicu recompose UI yang bergantung padanya secara otomatis.
  • add(0, ...) – Menyisipkan di indeks 0 agar produk terbaru tampil di paling atas daftar.

ui/screens/MainScreen.kt FR-03FR-04

Komponen Scaffold utama yang mengatur topBar, bottomBar, FAB, dan snackbar.

@Composable
fun MainScreen(viewModel: MarketplaceViewModel) {
    var currentScreen by remember { mutableStateOf("home") }
    val snackbarHostState = remember { SnackbarHostState() }
    val scope = rememberCoroutineScope()

    Scaffold(
        snackbarHost = { SnackbarHost(snackbarHostState) },
        topBar = { CenterAlignedTopAppBar(
            title = { Text(if (currentScreen == "add") "Tambah Produk" else "MarketSiswa") },
            navigationIcon = { /* tombol Back hanya di layar "add" */ }
        )},
        bottomBar = { NavigationBar {
            NavigationBarItem(selected = currentScreen == "home", ...)  // [FR-03]
            NavigationBarItem(selected = currentScreen == "profile", ...)
        }},
        floatingActionButton = {
            if (currentScreen == "home") ExtendedFloatingActionButton(...)  // [FR-02]
        }
    ) { innerPadding ->
        when (currentScreen) {
            "home"    -> HomeScreen(products = viewModel.products)
            "add"     -> AddProductScreen(onProductAdded = { name, price, desc ->
                viewModel.addProduct(name, price, desc)
                scope.launch {
                    currentScreen = "home"
                    snackbarHostState.showSnackbar("Produk berhasil ditambahkan!")  // [FR-04]
                }
            })
            "profile" -> ProfileScreen()
        }
    }
}
  • currentScreen – Navigasi sederhana berbasis string state; tidak perlu NavHost karena alur linear.
  • CenterAlignedTopAppBar – Judul berganti dinamis. Tombol Back muncul hanya di layar "add".
  • NavigationBar – Bottom navigation Material Design 3 dengan indikator tab aktif (FR-03).
  • ExtendedFloatingActionButton "Jual" – Hanya muncul di Home sebagai entry point penambahan produk (FR-02).
  • snackbarHostState.showSnackbar(...) – Dijalankan dalam coroutine (scope.launch) karena merupakan fungsi suspend (FR-04).

ui/screens/HomeScreen.kt FR-01

Menampilkan sapaan dan daftar produk dalam Card yang bisa di-scroll.

@Composable
fun HomeScreen(products: List<Product>) {
    LazyColumn(contentPadding = PaddingValues(16.dp), ...) {
        item { Text("Halo, Siswa!", fontWeight = ExtraBold) }
        items(products) { product -> ProductCard(product) }
    }
}

@Composable
private fun ProductCard(product: Product) {
    Card(shape = RoundedCornerShape(16.dp)) {
        Row {
            Text(product.name, fontWeight = Bold, fontSize = 18.sp)
            Text("Rp ${product.price}", color = primary, fontWeight = Black)
        }
        Text(product.description, color = DarkGray, fontSize = 14.sp)
    }
}
  • LazyColumn – Setara RecyclerView di Compose. Hanya merender item yang terlihat di layar untuk performa scroll mulus (Non-Functional: smooth scrolling).
  • items(products) – Menerima list langsung dari mutableStateListOf ViewModel; perubahan list memicu recompose otomatis tanpa collectAsState.
  • Card dengan RoundedCornerShape(16.dp) – Menerapkan prinsip Material Design: rounded corner + shadow memberi kesan elevasi (FR-01).
  • Hierarki teks (nama bold → harga berwarna → deskripsi abu-abu) menerapkan prinsip Visual Hierarchy dari PRD.

ui/screens/AddProductScreen.kt FR-02FR-04

Form input untuk menambah produk baru dengan simulasi loading.

@Composable
fun AddProductScreen(onProductAdded: (name, price, desc) -> Unit) {
    var name by remember { mutableStateOf("") }
    var price by remember { mutableStateOf("") }
    var desc  by remember { mutableStateOf("") }
    var isLoading by remember { mutableStateOf(false) }
    val scope = rememberCoroutineScope()

    OutlinedTextField(value = name, label = { Text("Nama Produk") }, ...)
    OutlinedTextField(
        value = price,
        keyboardOptions = KeyboardOptions(keyboardType = KeyboardType.Number), ...)
    OutlinedTextField(value = desc, minLines = 3, ...)

    Button(
        enabled = name.isNotBlank() && price.isNotBlank() && !isLoading,
        onClick = {
            isLoading = true
            scope.launch {
                delay(1000)                   // Simulasi proses [FR-04 State B]
                onProductAdded(name, price, desc)
                isLoading = false
            }
        }
    ) {
        if (isLoading) CircularProgressIndicator(modifier = Modifier.size(24.dp))
        else Text("Simpan Produk")
    }
}
  • OutlinedTextField – Input Material Design 3 dengan label animasi, mewakili "Text Field" dari PRD.
  • KeyboardType.Number – Memunculkan keyboard angka untuk field Harga, mencegah input teks tidak valid.
  • enabled = ... && !isLoading – Tombol Simpan hanya aktif bila nama & harga terisi dan tidak sedang loading; mencegah submit ganda.
  • delay(1000) – Simulasi latency proses selama 1 detik (FR-04 State B: Proses).
  • CircularProgressIndicator – Muncul menggantikan label tombol selama proses berlangsung (FR-04 loading indicator).

MainActivity.kt

Entry point Activity yang menginisialisasi Compose dan ViewModel.

class MainActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        enableEdgeToEdge()
        setContent {
            MarketplaceSiswaTheme {
                val viewModel: MarketplaceViewModel = viewModel()
                MainScreen(viewModel = viewModel)
            }
        }
    }
}
  • enableEdgeToEdge() – Konten meluas ke tepi layar untuk tampilan immersive modern.
  • viewModel() – Membuat atau mendapatkan kembali instance ViewModel yang bertahan selama lifecycle Activity, sehingga data produk tidak hilang saat rotasi layar.

Alur Pengguna (User Flow)

  1. Buka app → HomeScreen menampilkan "Halo, Siswa!" dan dua produk awal dalam Card (FR-01).
  2. Ketuk FAB "Jual" → layar beralih ke AddProductScreen dengan judul "Tambah Produk" (FR-02).
  3. Isi form → Simpan Produk → tombol menampilkan CircularProgressIndicator selama 1 detik (FR-04 loading), lalu kembali ke Home.
  4. Snackbar "Produk berhasil ditambahkan!" muncul dan produk baru tampil paling atas (FR-04 snackbar).
  5. Tab "Profile" → ProfileScreen menampilkan John Siswa / XII RPL 1 (FR-03).
